Array & ArrayList

Array

An array is a data structure in Java that can store many values of the same type. For example, it can store integers, floats, Strings, or classes. Arrays allow for a group of data to be accessed with 1 variable name, the n ame of the array.

Declaring an Array

This is how an array is declared: 

      type[] name;

The array can be initialized like this: 

      type[] name = new type[size];
    
      int[] arr = new int[4];

This means that a new int array called 'arr' is created, with 4 indices.

Alternatively, an array can be initialized like this 

      int[] arr = { 3, 4, 6, 7 };

Accessing values in an array

Values in an array can be accessed like so:name[index] = value; 

      System.out.println(arr[0]);

will print 3

The values in an array can be changed by assigning a value to a specific index. name [index] = value;

So, to change the value 4 in the array to 9, we can do arr[1] = 9;

Arrays have a static size, meaning that the size cannot be changed after it is initialized. This is useful to know in an example of inserting a value at an index. This code shows how to insert the value 10 in between the numbers 6 and 7 in the array above. 

      
    int [] otherArr = new int [5]; // the new array will be 1 index longer than arr

    otherArr[0] = arr[0]; // copies the value over

    otherArr[1] = arr[1]; // copies the value over

    otherArr[2] = arr[2]; // copies the value over

    otherArr[3] = 10; // between arr[2] and arr[3], add the value 10

    otherArr[4] = arr[3]; // copies the value over

Now the array otherArr has the values {3,9,6,10,7}

ArrayList

ArayLists are similar to arrays because they are also used to store values that can be referred to with 1 name. The main difference between the two is the dynamic size of ArrayLists. Before using an ArrayList,java.util.ArrayList must be imported.

Declaring and initializing an ArrayList 

      ArrayList  name = new ArrayList();
      ArrayList arrList = new ArrayList();

This creates a new Integer ArrayList called arrList. Values can be added to the ArrayList using the add() method. 

      arrList.add(1);

      arrList.add(4);

This adds the Integer 1 and 4 to the list, so the contents of arrList are now{1,4}and the size of arrList is 2. Accessing values in an ArrayList

Values can be accessed with theget()method. 

      arrList.get(0);

will return 1.
