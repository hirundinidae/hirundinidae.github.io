Object Oriented Programming

What is Objet Oriented Programming?

There are 4 main models of programming, Imperative, Functional, Logical, and Object Oriented.

Object Oriented Programming, or OOP, is a style of programming that uses the idea of 'objects'. These objects have data or information about them, such as it's characteristics and abilities. Objects can interact with other objects, just like in real life. A person is an object, and a person can interact with other people

For example: an animal is an object. An animal can interact with the environment around them, such as other animals, and other entities. An animal also has capabilities, such as eating, drinking, and sleeping, and characteristics, like number of legs, diet, size, etc.
